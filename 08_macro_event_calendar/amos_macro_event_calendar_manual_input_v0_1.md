# AMOS Macro Event Calendar Manual Input v0.1

这个交付建立 AMOS 的手动宏观/事件日历。

## 覆盖事件

- CPI / PCE
- NFP / JOLTS / Labor
- FOMC / FOMC Minutes / Fed Speakers
- Treasury Auctions
- OPEX / Quarterly OPEX
- Earnings / Company Events
- AI Industry Events
- BTC / Crypto Events
- Political / Geopolitical Events

## 工作簿包含

- Overview
- Event Calendar
- Weekly Risk View
- Event Categories
- Official Sources
- Lists

## 核心规则

Critical events 默认进入 **No Attack**。  
除非 setup 已经确认，且风险明确接受，否则不在事件前主动猛攻。
