# AMOS Real Market Data Source Map v0.1

这个交付把 AMOS CSV 字段映射到真实/手动可取的数据来源。

## P0 数据源

- TradingView Chart Data CSV
- TradingView Screener Export CSV
- SEC EDGAR APIs
- Company Investor Relations
- Nasdaq Historical / Market Activity

## 仍未最终选择的数据源

- Options / Gamma provider
- BTC / crypto market data provider
- Macro event calendar standard source

## 工作簿包含

- Overview
- Source Map
- Field to Source
- Import Plan
- Source Quality
- Lists

## 护栏

Data Source Map 只是定义来源标准，不绕过：

- Evidence Packet
- Anchor Verification
- Action Cap
- Live price / AVWAP / RSI / TCG confirmation
