Attack

allowed when

  • Data Freshness is Fresh or high-quality Partial
  • Action cap says No Cap
  • Macro does not block beta
  • RS vs QQQ and RS vs peers are improving
  • Technical confirmation exists
  • Invalidation is clear

forbidden when

  • Any P0 must-have data missing
  • No invalidation
  • Gann only signal
  • Event pressure high without confirmation
  • User is chasing missed move

Build

allowed when

  • Action cap at least Build/Hold Only
  • Setup has confirmation
  • Panic maturity or trend continuation is valid
  • Position size is staged

forbidden when

  • Risk Stack high and still rising
  • AVWAP lost
  • RS deteriorating
  • No stop/invalidation

Hold

allowed when

  • Thesis intact
  • Structure intact
  • No major invalidation triggered
  • Risk acceptable

forbidden when

  • Fundamental thesis broken
  • Structure shifts to LH/LL
  • AVWAP lost with failed reclaim
  • Evidence packet rejected

Trim Watch

allowed when

  • Risk Stack high
  • Narrative maturity high
  • Gann resistance/time window active
  • RSI overbought or volume exhaustion
  • Event pressure elevated

actual trim requires

  • AVWAP/EMA break
  • Failed new high / LH appears
  • Volume divergence
  • Good-news fade
  • Gamma/event pressure confirms

Wait

required when

  • Data missing
  • Action cap says Wait Only
  • No setup
  • Only narrative excitement exists
  • User observation lacks evidence packet

No-Trade Rules

  • No leverage.
  • No action without invalidation.
  • No chasing after missed move.
  • Gann cannot act alone.
  • Headline dip is not panic maturity.
  • Do not turn failed swing into core.
  • Do not let one blogger/source override data.
  • If data freshness is Missing, no aggressive action.

Weekend Workflow

  • Review Prediction → Action → Result.
  • Update Evidence Packet status.
  • Re-score source quality.
  • Review which tickers deserve P0/P1/P2 priority.
  • Prepare next week's event calendar.
  • Clean watchlist to avoid attention dilution.
