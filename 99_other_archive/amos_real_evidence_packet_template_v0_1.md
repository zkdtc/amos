# AMOS Real Evidence Packet Template v0.1

这个交付在手动 Alpha 工作簿中新增：

- `Evidence Packet v0.1`
- `Evidence Packet Guide`

## 目的

每一个观点、新闻、博主结论、财报结论、用户观察，进入 AMOS 前都必须变成 Evidence Packet。

## Evidence Packet 字段

- Packet ID
- Date
- Ticker(s)
- Narrative(s)
- Source Type
- Source Name / URL
- Author / Speaker
- Claim
- Claim Type
- Time Horizon
- Module Target
- Bull / Bear / Neutral
- Evidence Quality
- Bias Risk
- Freshness
- Specificity
- Actionability
- Counter Evidence Required?
- Counter Evidence
- What Would Prove It Wrong?
- Linked Data Fields Needed
- Decision Impact
- Suggested AMOS Action
- Confidence Cap
- Can Affect Position?
- Review Date
- Status
- Notes
- Final Evidence Score
- Action Gate

## 评分公式

```text
Evidence Score =
30% Evidence Quality
+ 15% (100 - Bias Risk)
+ 15% Freshness
+ 15% Specificity
+ 15% Actionability
+ 10 bonus if counter evidence is required
```

## Action Gate

- >=75：Can Influence Action
- 60-74：Can Influence Watch
- 45-59：Research Only
- <45：Do Not Use

## 核心规则

用户观察是线索，不是证据本身。  
博主观点是框架线索，不是市场事实。  
官方数据、价格行为、财报、成交量和反证共同决定权重。
