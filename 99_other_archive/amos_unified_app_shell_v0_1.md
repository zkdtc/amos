# AMOS Unified App Shell v0.1

这是 AMOS 的统一产品壳，把此前分散的模块整合成一个导航式单页入口。

## 作用

- 统一入口
- 左侧导航
- 顶部系统状态
- 模块卡片
- 工程路线
- 让 AMOS 从“一堆文件”开始变成“产品结构”

## 当前状态

当前仍然是 Static Alpha，不是数据驱动版本。

已整合模块：

1. Home Command Center
2. CRDO Stock Intelligence Page
3. Macro Regime
4. Narrative & Rotation
5. Portfolio Migration
6. Risk Guardrail
7. Valuation Battle
8. Gann × Event × Gamma
9. Stock Personality
10. Evidence Quality
11. Data Schema
12. Agent Routing
